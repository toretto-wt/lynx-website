# lynx-devtool

让 agent 与 Lynx DevTool 交互，在已连接的设备（Android、iOS、桌面端）上检查和调试运行中的 Lynx 应用：

- **CDP 命令** — 发送 Chrome DevTools Protocol 命令，用于 DOM 检查、CSS 查询、控制台日志获取和截图。
- **App 命令** — 在已连接的 Lynx 应用中打开或关闭页面。
- **打开 URL** — 直接在 Lynx 中打开 URL 进行开发和测试。
- **故障排查** — 内置 agent 与 DevTool 之间的连接器和传输问题诊断。

## 安装

```bash
npx skills add lynx-community/skills -s lynx-devtool
```

这会将 skill 规则添加到你的项目中，以便兼容的 coding agent（如 Claude Code、Cursor 或 Copilot）能够自动应用它们。
