# WheelEvent

<APISummary />

表示鼠标滚轮事件，继承自 [`MouseEvent`](./mouse-event)。触摸板或其他滚动设备也可以触发该事件。[`target`](./event#target) 是第一次触发 `WheelEvent` 时，鼠标指针所在的 `element`。

## 实例属性

### deltaX

```ts
deltaX: number;
```

表示横向滚动距离，单位为逻辑像素。

### deltaY

```ts
deltaY: number;
```

表示纵向滚动距离，单位为逻辑像素。

## WheelEvent 的类型

### wheel

表示用户滚动鼠标滚轮或触摸板。[`target`](./event#target) 是第一次触发 `WheelEvent` 时，鼠标指针所在的 `element`。
