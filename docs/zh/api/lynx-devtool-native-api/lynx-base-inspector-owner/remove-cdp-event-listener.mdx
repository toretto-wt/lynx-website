# removeCDPEventListener

<APISummary />

移除一个 [CDP](https://chromedevtools.github.io/devtools-protocol/) **Event** 消息的监听器。

此方法是 [`addCDPEventListener`](/api/lynx-devtool-native-api/lynx-base-inspector-owner/add-cdp-event-listener) 的配对操作。当你不再需要监听消息时，应调用此方法来移除监听器。

## 语法

### iOS

```objective-c title="LynxBaseInspectorOwner.h"
- (void)removeCDPEventListener:(nonnull NSString *)name;
```

### Android

```java title="LynxBaseInspectorOwner.java"
void removeCDPEventListener(final String name);
```

### 参数说明

- `name`: listener 的唯一标识符，与 [`addCDPEventListener`](/api/lynx-devtool-native-api/lynx-base-inspector-owner/add-cdp-event-listener#参数说明) 中传入的 `name` 一致。

## 使用示例

请参考 [`CDPEventListener`](/api/lynx-devtool-native-api/cdp-event-listener#使用示例)。
