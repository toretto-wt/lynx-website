# setGlobalPropsUpdatedObserver

<APISummary />

注册 GlobalProps 的观察者，用于监听 GlobalProps 的更新。

## 语法

### iOS

```objective-c title="LynxBaseInspectorOwner.h"
- (void)setGlobalPropsUpdatedObserver:(id<GlobalPropsUpdatedObserver>)observer;
```

### Android

```java title="LynxBaseInspectorOwner.java"
void registerGlobalPropsUpdatedObserver(GlobalPropsObserver observer);
```

### 参数说明

- `observer`: 实现了 [`GlobalPropsUpdatedObserver`](/api/lynx-devtool-native-api/global-props-updated-observer) 的对象。

## 使用示例

请参考 [`GlobalPropsUpdatedObserver`](/api/lynx-devtool-native-api/global-props-updated-observer#使用示例)。
