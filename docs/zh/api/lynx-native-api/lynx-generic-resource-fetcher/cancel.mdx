# cancel

<APISummary />

取消请求所需的资源。

## 语法

### Android

```java
public void cancel(LynxResourceRequest request);
```

#### 参数说明

- `request`: 必需的请求。

:::note

- 这个方法是可选实现的。

:::

### Harmony

```typescript
abstract cancel(request: LynxResourceRequest): void;
```

#### 参数说明

- `request`: 要取消的请求。

### Desktop (C++)

Desktop public API 通过请求 ID 取消资源获取：

```cpp
virtual void Cancel(lynx_resource_request_id request_id);
```

#### 参数说明

- `request_id`: 要取消的请求 ID。
