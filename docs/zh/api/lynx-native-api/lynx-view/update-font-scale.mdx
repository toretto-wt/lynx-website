# updateFontScale

<APISummary />

在客户端设置中更改字体缩放比例将自动更改文本大小。

## 语法

### Android

```java
public void updateFontScale(float scale);
```

#### 参数说明

- `scale`: 新的字体缩放比例。

### iOS

```objc
- (void)updateFontScale:(CGFloat)scale;
```

#### 参数说明

- `scale`: 新的字体缩放比例。

### Desktop (C++)

```cpp
void SetFontScale(float scale);
```

#### 参数说明

- `scale`: 新的字体缩放比例。
