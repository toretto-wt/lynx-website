# onFirstScreen

<APISummary />

首屏布局完成时调用。

## 语法

### Android

```java
public void onFirstScreen();
```

### iOS

```objc
- (void)lynxViewDidFirstScreen:(LynxView *)view;
```

#### 参数

- `view`: LynxView 实例。

:::note

- 此时您可以获取 LynxView 加载过程中的性能数据。

:::

### Harmony

```typescript
public onFirstScreen(): void;
```

### Desktop (C++)

```cpp
class YourViewClient : public lynx::pub::LynxViewClient {
 public:
  void OnFirstScreen() override;
};
```
