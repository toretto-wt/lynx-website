# onDataUpdated

<APISummary />

数据更新时调用，但视图可能尚未更新。

## 语法

### Android

```java
public void onDataUpdated();
```

### iOS

```objc
- (void)lynxViewDidUpdate:(LynxView *)view;
```

#### 参数

- `view`: LynxView 实例。

### Harmony

```typescript
public onDataUpdated(): void;
```

### Desktop (C++)

```cpp
class YourViewClient : public lynx::pub::LynxViewClient {
 public:
  void OnDataUpdated() override;
};
```
