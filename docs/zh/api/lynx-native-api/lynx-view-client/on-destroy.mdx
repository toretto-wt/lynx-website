# onDestroy

<APISummary />

页面销毁后调用。

## 语法

### Android

```java
public void onDestroy();
```

:::note

- 回调在主线程中执行。

:::

### Harmony

```typescript
public onDestroy(): void;
```

### Desktop (C++)

```cpp
class YourViewClient : public lynx::pub::LynxViewClient {
 public:
  void OnDestroy() override;
};
```
