# setGlobalPropsUpdatedObserver

<APISummary />

Registers an observer for GlobalProps to listen for updates.

## Syntax

### iOS

```objective-c title="LynxBaseInspectorOwner.h"
- (void)setGlobalPropsUpdatedObserver:(id<GlobalPropsUpdatedObserver>)observer;
```

### Android

```java title="LynxBaseInspectorOwner.java"
void registerGlobalPropsUpdatedObserver(GlobalPropsObserver observer);
```

### Parameters

- `observer`: An object that implements [`GlobalPropsUpdatedObserver`](/api/lynx-devtool-native-api/global-props-updated-observer).

## Usage Example

For an example, see [`GlobalPropsUpdatedObserver`](/api/lynx-devtool-native-api/global-props-updated-observer#usage-example).
