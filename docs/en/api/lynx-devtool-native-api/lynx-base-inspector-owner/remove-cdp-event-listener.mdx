# removeCDPEventListener

<APISummary />

Removes a listener for [CDP](https://chromedevtools.github.io/devtools-protocol/) **Event** messages.

This method is the counterpart to [`addCDPEventListener`](/api/lynx-devtool-native-api/lynx-base-inspector-owner/add-cdp-event-listener). You should call it to remove the listener when you no longer need to listen for messages.

## Syntax

### iOS

```objective-c title="LynxBaseInspectorOwner.h"
- (void)removeCDPEventListener:(nonnull NSString *)name;
```

### Android

```java title="LynxBaseInspectorOwner.java"
void removeCDPEventListener(final String name);
```

### Parameters

- `name`: The unique identifier of the listener, consistent with the `name` passed in [`addCDPEventListener`](/api/lynx-devtool-native-api/lynx-base-inspector-owner/add-cdp-event-listener#parameters).

## Usage Example

Please refer to [`CDPEventListener`](/api/lynx-devtool-native-api/cdp-event-listener#usage-example).
