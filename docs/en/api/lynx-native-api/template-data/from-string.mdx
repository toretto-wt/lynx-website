# fromString

<APISummary />

Input data in json and return the parsed `TemplateData` object.

## Syntax

### Android

```java
public static TemplateData fromString(String json);
```

#### Parameters

- `json`: `String` type data in json format.

#### Return

- `TemplateData` constructed by `json`.

### iOS

```objc
- (instancetype)initWithJson:(NSString *)json;
```

#### Parameters

- `json`: `NSString*` type data in json format.

#### Return
- `TemplateData` constructed by `json`.

### Desktop (C++)

Desktop public API constructs `LynxTemplateData` directly from the JSON string:

```cpp
auto data = std::make_shared<lynx::pub::LynxTemplateData>(json);
```

#### Parameters

- `json`: The JSON string used to create the template data.
