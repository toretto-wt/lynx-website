## [Announcing Lynx for HarmonyOS](/blog/lynx-harmony)

_Sep 25, 2025 by [Qiang Bai](https://github.com/rel-q)、[Wentao Shi](https://github.com/toretto-wt), representing [The Lynx Team](https://x.com/lynxjs_org)_

Today, we are happy to share that Lynx now provides public beta support for HarmonyOS, delivering feature parity in CSS, Elements, and APIs — along with the same level of native performance and user experience.

## [Lynx 3.4: HarmonyOS Support, Trace and Recorder, Text Input Elements](/blog/lynx-3-4)

_September 3rd, 2025 by [Leron Liu](https://github.com/loongliu), representing [The Lynx Team](https://x.com/lynxjs_org)_

Lynx 3.4 is now officially released!
This release features new platform support for HarmonyOS, enables Windows development, launches new developer tools _Trace_ and _Recorder_, open-sources the highly requested `<input>` and `<textarea>` elements, offers even more flexible animations, supports variable fonts, and more.

## [What’s new in Lynx 3.2](/blog/lynx-3-2)

_May 14th, 2025 by [Wentao Shi](https://github.com/toretto-wt), representing [The Lynx Team](https://x.com/lynxjs_org)_

Lynx 3.2 delivers updates across its framework, engine, and tools, including the ReactLynx testing tools, `llms.txt` support, new Grid Layout features, improvements to `<list>` and `<text>`, and more.
