# lynx-typescript

This article discusses common TypeScript issues and solutions in Lynx development, including environment configuration, type extension, event handling, LynxUI components, and advanced ReactLynx usage.

## Installation

```bash
npx skills add lynx-community/skills -s lynx-typescript
```

This adds the skill rules to your project so that compatible coding agents (such as Claude Code, Cursor, or Copilot) can automatically apply them.
